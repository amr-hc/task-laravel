## Getting Started

1. **Navigate to the project directory:**
    ```bash
    cd your-project
    ```

2. **Install dependencies:**
    ```bash
    composer install
    ```

3. **Run migrations:**
    ```bash
    php artisan migrate
    ```

4. **Start the server:**
    ```bash
    php artisan serve
    ```

## Background Tasks

- **Run scheduled tasks:**
    ```bash
    php artisan schedule:run
    ```

- **Process queue jobs:**
    ```bash
    php artisan queue:work
    ```
